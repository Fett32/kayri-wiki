**Chipped** adds 7 specialized workbenches, each one giving you dozens of decorative variants for a family of vanilla blocks. 1 input block = 1 output variant — no extra material cost. It also adds **handheld tools** so you can swap a placed block's variant without breaking it.

## The seven workbenches

### Glassblower

All glass + glass panes (plain and stained — every color).

**Recipe:** blast furnace + 4 bricks + 2 iron + 1 glass + 1 crafting table.

### Carpenter's Table

All wood — planks, logs, stripped logs, doors, trapdoors. Also barrel, bookshelf, ladder, and torch variants.

**Recipe:** iron axe + iron ingot + 5 planks + 1 crafting table.

### Mason Table

All stone family — cobblestone, granite, diorite, andesite, deepslate, blackstone, basalt, tuff, calcite, dripstone, end stone, quartz, prismarine, obsidian — plus terracotta (plain and glazed, every color), concrete (every color), brick variants, sandstone, raw copper/iron/gold blocks, ancient debris.

**Recipe:** 3 bricks + 2 iron + 3 logs + 1 crafting table.

### Loom Table

All wool + all carpet.

**Recipe:** 3 wool + 2 sticks + 3 logs + 1 crafting table.

### Botanist Workbench

Organic blocks — all leaves, mushroom blocks/stems, ice/packed ice/blue ice, snow, dirt/mud/clay, pumpkins (incl. carved + jack o'lantern), froglights, sand/soul sand/gravel, moss, lily pads, vines, nether plants, dried kelp, mangrove roots, hay, melon, cobweb.

**Recipe:** 3 flower pots + 2 wooden slabs + 2 sticks + 1 crafting table.

### Alchemy Bench

Fancy material blocks — diamond, emerald, gold, iron, netherite, lapis, amethyst, coal blocks, glowstone, lodestone, crying obsidian, sponge, all 4 waxed copper oxidation stages.

**Recipe:** brewing stand + enchanting table + 4 wooden slabs + 1 crafting table.

### Tinkering Table

Redstone and light blocks — lanterns (regular, soul, special, special soul), sea lantern, redstone lamp, redstone block, redstone torch, iron bars.

**Recipe:** 2 redstone torches + 1 redstone + 2 iron + 2 logs + 1 crafting table.

## Handheld tools

Each bench (except Glassblower) has a matching tool. With a tool in hand:

- **Shift + right-click** a placed block to cycle through variants.
- **Right-click** to apply the current variant to a block.

| Tool | Bench | Extra recipe cost |
|---|---|---|
| Chisel | Mason Table | + iron ingot |
| Saw | Carpenter's Table | + iron ingot |
| Needles | Loom Table | + stick |
| Watering Can | Botanist Workbench | + bucket |
| Multimeter | Tinkering Table | + redstone |
| Alchemy Book | Alchemy Bench | + book |

Glass swaps happen at the bench itself — no handheld tool.

## Practical tips

- **Any bench accepts any variant from its family as input.** Dropping plain cobblestone in the Mason Table lets you pick from every stone variant — not just cobble-derived ones.
- **Variants are 1-for-1**, no extra materials. It's a decorator, not a resource sink.
- **Handheld tools are great for last-minute aesthetic tweaks** — change a wall's wood type without breaking and replacing blocks.
- **Combine with Carpenter's Table** on planks for endless wood paneling/trim per wood type.
