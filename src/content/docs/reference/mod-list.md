- **No duplicates or version conflicts detected** as of snapshot date.

Many vanilla/mod-bundled datapacks are toggled on by default via `initial-enabled-packs`. Notable ones:
- All `mr_*` packs (Moonrise Modrinth bundles — ancient cities, grand capitals, strongholds, sleep, true ending, wilderness)
- `mvs` (Moog's Voyager Structures)
- All `yungs*` datapacks
- Rearm's `magic_protection`, `backstep`, `crippling_throw`, `bash`
- Enderscape's `new_end_cities`, `new_terrain`, `fix_vanilla_recipes`
- `t_and_t:resources/t_and_t_waystones_patch`
- `cristellib:runtime_pack` starts **disabled** (`initial-disabled-packs`)
