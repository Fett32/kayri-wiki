## Trigger

Ghost-type Pokémon

## What it does

Harvests netherwart from soul sand. **Automatically replants.**

## Cadence

Passive — as netherwart matures

## Setup requirements

Netherwart patches in scan zone

## Good Pokémon for this job

- Gastly → Haunter → Gengar
- Duskull → Dusknoir
- Shuppet → Banette
- Any Ghost-type

## Tips

- Place the pasture in a Nether outpost for bulk netherwart
- Works in overworld too if you've set up netherwart on soul sand
