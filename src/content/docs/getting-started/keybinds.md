Quick reference for what's bound in the distributed pack. Everything not listed here is vanilla MC default.

## Changed from default (moved to avoid conflicts)

| Key | Function | Was |
|---|---|---|
| **J** | Traveler's Backpack: open inventory | *B* |
| **K** | Xaero: open full map | *M (Iris toggleShaders was here, now unbound)* |
| **'** | Where Is It: search | *Y* |
| **,** | Xaero: new waypoint | *B* |

## Preserved non-default bindings worth knowing

| Key | Function |
|---|---|
| **B** | Full Bright toggle |
| **M** | Cobblemon: Pokémon Summary |
| **R** | Cobblemon: Throw / Recall Party Pokémon |
| **O** | Cobblemon: Hide Party |
| **L.Alt** | Cobblemon: Riding Freelook (also used for Raid Mouse — context-gated) |
| **G** | Fight or Flight: Start Battle |
| **V** | Supplementaries: Quiver |
| **H** | Accessories: Open Screen |

## Power-user tool configs

Moved to avoid overlap with the above:

- **Litematica** chord modifier: **Right Control** *(was M — conflicted with Pokémon Summary)*
  - Schematic Menu: `R.Ctrl + A`
  - Material List: `R.Ctrl + L`
  - etc.
- **MiniHUD** config GUI: **F8** *(was H — conflicted with Accessories)*

## Left unbound (set your own in Options → Controls)

- Iris: Reload Shaders, Shader Pack Selection, Toggle Shaders
- Rearm: Action
- Ping Wheel: Name Labels
- Elixirum: Menu
- Xaero: Minimap Settings
- Simply Tooltips: Capture GIF
- Weapon Master: Open GUI

## Notes

- Vanilla MC defaults (movement, inventory **E**, chat **T**, etc.) are unchanged.
- Shared keys like **L.Shift** (sneak + tooltip scroll) and middle-click (pickItem + inv sort) are fine — they fire in different contexts.
- Anything weird — just rebind in **Options → Controls**.
