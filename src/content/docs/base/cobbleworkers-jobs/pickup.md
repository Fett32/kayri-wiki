## Trigger

Any Pokémon with the **Pickup** ability

## What it does

Rolls pickup loot tables — Poké Balls, evolution stones, type gems, ability capsules, EXP candy, apricorn seeds, and more.

## Cadence

Every **120 seconds** per pasture

## Setup requirements

Just the Pokémon in the pasture — nothing else required

## Good Pokémon for this job

- Meowth / Persian
- Zigzagoon / Linoone
- Teddiursa / Ursaring
- Pachirisu
- Bunnelby / Diggersby
- Shroomish
- Pansage / Pansear / Panpour
- Anything with Pickup

## Tips

- Multiple Pickup mons in one pasture = one roll per pasture, not per mon
- Spread Pickup mons across multiple pastures for more rolls
- Loot tables include: ancient held items, common Poké Balls, evo stones, EXP candy, natural heal items, type gems, apricorn seeds, village loot
