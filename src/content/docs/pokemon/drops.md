Vanilla mobs are gone in all dimensions, so **Pokémon drop the items you'd normally get from mobs**. If you need something, find which Pokémon to fight. This is a quick-reference page — see the [full drops lookup](/reference/drops-lookup/) for every species.

## Food: meat

| Item | Source Pokémon |
|---|---|
| **Beef** | Miltank, Tauros, Bouffalant, Stantler, Deerling, Sawsbuck, Girafarig |
| **Porkchop** | Tepig line, Spoink line, Swinub line, Phanpy line, Lechonk, Oinkologne |
| **Chicken** | Pidgey, Spearow, Fletchling line, Pidove line, Ducklett, Squawkabilly, Wingull + more birds |
| **Mutton** | Mareep line, Wooloo line + most Normal-types |
| **Rabbit** | Buneary / Lopunny, Bunnelby / Diggersby, Azumarill |
| **Cod / Salmon** | Most Water-types |

## Especially notable Pokémon

- **Miltank** — beef + leather + milk (vanilla farm-animal replacement in one mon)
- **Chansey line** — the egg source
- **Dragonite** — Dragon Breath, Amethyst, Feather, Phantom Membrane, Scute
- **Cubone / Marowak** — bones, and Marowak's rare Wither Skull

## Drop mechanics

- Drops go **directly to your inventory** — not on the ground
- You see what dropped in chat
- Drops appear immediately when a Pokémon faints

No waiting, no scramble, no despawn timer.
